# 🔦 PyTorch Deep Learning Projects

A collection of projects implemented using Pytorch

---

## 📂 Repository Structure
- [MNIST](MNIST/): Implementation of a Convolutional Neural Network (CNN) to classify handwritten digits. This project covers the basics of torch.nn, data loaders, and the training loop (optimizer, loss function, backpropagation).

- [housing_dataset](housing_dataset/): A regression task focusing on predicting household values. This project explores data normalization, handling tabular data with Tensors, and evaluating model performance using MSE.

More detailed descriptions are inside the folders.
